# 🧾 Multi-Step Form UI

A clean and simple **Multi-Step Form UI** built using **HTML, CSS, and Vanilla JavaScript**.

This project demonstrates how large forms can be divided into **multiple steps** to improve user experience. Users can navigate between steps using **Next** and **Previous** buttons, and all form inputs are collected and processed using JavaScript.

The project is created as a **frontend learning exercise** to practice DOM manipulation, form handling, and UI structuring.

---

## 🧱 Project Structure

```bash
Multi-Step-Form-UI/
│
├── index.html          # Form structure and layout
├── app.css             # UI styling and layout design
├── main.js             # Step navigation and form logic
├── LICENCE
└── README.md           # Project documentation
```

---

## ✨ Features

### 🪜 Multi-Step Form Navigation

* Step-by-step form workflow
* Navigate using **Next** and **Previous** buttons
* Only one step visible at a time

### 📋 Form Data Collection

* Collects all input values from the form
* Data stored in a JavaScript array
* Output logged to the browser console

### 🔄 Form Reset Handling

* Displays confirmation alert on submission
* Resets the form after submission
* Automatically returns to the first step

### 🎨 Clean UI Layout

* Minimal form design
* Card-style container layout
* Simple and readable input fields
